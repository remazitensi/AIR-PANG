 <h1> :cloud: AIR PANG : 대기질 정보 공유 및 개선 챌린지 서비스 </h1> 

![image](https://github.com/user-attachments/assets/b2a8ffbb-f7f9-4d8c-9038-b5aece84fd02)

## 프로젝트 소개

- 공기 팡은 사용자에게 실시간 대기질 정보를 제공합니다.
- 지역별 대기 오염 수준을 시각적으로 비교할 수 있습니다.
- 사용자는 자신의 지역 대기질을 확인하고 친환경 챌린지에 참여할 수 있습니다.
- 과거 대기질 데이터와 비교해 변화에 대한 인사이트를 얻고, 더 나은 환경을 위한 행동을 유도합니다.
